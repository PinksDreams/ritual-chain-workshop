# Reflection

In a bounty system, the rules, deadlines, reward amount, commitments, and final winner should be public because participants need to verify that the process is fair. The actual answers should stay hidden during the submission phase, because public answers create bad incentives and allow later participants to copy earlier work. Private reasoning, sensitive metadata, and encrypted submissions should also stay hidden until they are needed for judging. AI can be useful for comparing submissions, summarizing strengths and weaknesses, and producing an initial ranking. However, humans should still define the rules, handle disputes, and make final decisions in unclear or high-value cases. The best system is not one where AI controls everything, but one where AI helps evaluate work inside clear boundaries. Public data should create accountability, while hidden data should protect fairness before the deadline. Commit-reveal is a simple way to separate those two needs.
